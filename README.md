# Luma AI

Luma AI is a conversational AI chatbot built with **Python**, **Streamlit**, **LangChain**, and **Mistral AI**. It provides a clean chat interface with conversation memory, allowing users to interact naturally with the assistant.

## Features

- Interactive chat interface
- Conversation memory
- Powered by Mistral Large
- Simple and responsive Streamlit UI
- Secure API key management using environment variables

## Tech Stack

- Python
- Streamlit
- LangChain
- Mistral AI
- python-dotenv
